Welcome!

I've been building this website to show off some fun widgets I've made!
I've used Django as a Python web framework and have deployed the site on an AWS LightSail instance.
You can check out the code on Git here - Note: it's still very much a work in progress and I plan to add to it as the ideas come to me.

Just a note - I've built these apps primarily to be viewed on desktop. Some things won't work on mobile such as the video in the A* search or the PDF viewer on the resume page.

Stocks

Track the closing price movement of any stock available on Yahoo! Finance. The app also includes output of a trading simulation based on exponential moving average prices over the past year.

Twitter

Enter a search term and read the most recent tweets containing that term! Up to 100 tweets allowed. Can only provide tweets available via Twitters API.

A* Search Visualisation

I've built a Pygame application to visualise the movement of the A* search tree. Check this module out for links to the repo or feel free to download the executable file to run it yourself.
